## Совместимость плагина КриптоПро с браузерами

### ❌ Google Chrome
- **НЕ РАБОТАЕТ** с плагином КриптоПро
- Начиная с версии 137 Chrome прекратил поддержку расширений для ЭП
- Плагин не будет работать в Chrome

### ✅ Mozilla Firefox
- **РАБОТАЕТ** с плагином КриптоПро
- Поддержка включена по умолчанию
- Нужно разрешить плагин в настройках `about:addons`

### ✅ Chromium GOST
- **РАБОТАЕТ** с плагином КриптоПро
- Специальная сборка для работы с ЭП
- Поддерживает российские криптографические алгоритмы
- Можно скачать с сайта КриптоПро

---

## Рекомендация: Установить Firefox

### Установка Firefox

```bash
sudo apt update
sudo apt install -y firefox
```

### Настройка плагина в Firefox

1. Откройте Firefox
2. Перейдите на страницу: `about:addons`
3. Найдите плагин КриптоПро (должен быть автоматически обнаружен)
4. Разрешите его использование (если требуется)
5. Перезапустите Firefox

### Проверка работы

1. Вставьте токен Рутокен
2. Откройте сайт госуслуг
3. Нажмите "Войти с помощью ЭП"
4. Браузер должен предложить выбрать сертификат

## Альтернатива: Chromium GOST

Если нужен браузер на базе Chromium:

1. Скачайте Chromium GOST с сайта КриптоПро
2. Или установите из репозитория (если доступен)
3. Плагин должен работать автоматически
