## Описание проблемы

При попытке входа в личный кабинет с помощью электронной подписи возникают следующие проблемы:

1. **При входе через "Войти с помощью ЭП" на сайте ФНС:**
   - Диалог выбора сертификата открывается
   - Сертификат отображается в списке
   - Кнопка "Выбрать" не реагирует на клики мыши
   - Попытки использовать клавиатуру (Enter, Tab) также не помогают

2. **При входе через "Войти через Госуслуги (ЕСИА)":**
   - Открывается страница esia.gosuslugi.ru/login/
   - Появляется диалог "Проверка сертификатов"
   - Процесс поиска и проверки сертификатов уходит в бесконечный цикл
   - Диалог не закрывается, сертификат не выбирается
